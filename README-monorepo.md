# vitessce

## Usage

```sh
npm install vitessce
```

## Development

Install pnpm v7

### Setup

```sh
git clone 
pnpm install
```

### Run demo

```sh
pnpm run build # for @vitessce/icons and @vitessce/workers
pnpm run start
```

### Lint and format

```sh
pnpm lint
pnpm format
```

### Build library

```sh
pnpm build
```

### Build demo

```sh
pnpm build-demo
```

### Monorepo tasks

```sh
pnpm meta-updater
```
